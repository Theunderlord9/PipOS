# PipOS
This Is The PipOS From The Fallout Universe Made For A Humble Pi3B+ With A 3.5" Screen
It Needs Somewhere For SDL2 To Point To So I Recommend An X Layer as That Is What I Use

## Features
Stats Display PC Information
Radio (Not Complete) Plays Music read Its Documentation
Holotape Loading (A .py With The Holotape Class More In its Section) From The Inv Screen

## How Holotapes Work
Holotapes Are Simply Just A .py Script In The Holotapes Folder
### Requirements
They Must Have A Holotape Class And Pygame Imported
Init Is With '__init__(self, OverlaySurface, font)'
A Main(Screen) Where The Main Loop Must Be Run From
Main Needs To return Either Another App (.py File) To Be Loaded Or ShutDownOS However Most Should End With pipos.py As That Is Where The User Should End The OS From
